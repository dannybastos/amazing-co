# Amazing Co

A springboot application for the TradeShift challenge;

## What are we using here
- Spring Boot
- Swagger
- Actuator
- MockMVC

## System Requirements
- docker
- docker-compose

## How to run

```
docker-compose up -d
```

You can access the swagger in http://localhost:8080/swagger-ui.html

> The BootStrap class was created just to perform a Load Test.
